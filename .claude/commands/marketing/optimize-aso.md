## Platform-Specific Best Practices

**iOS App Store**:
- Update app every 2-4 weeks (signals active development)
- Use all 10 screenshot slots
- Subtitle is highly weighted for search (use keywords)
- Keyword field: Use singular OR plural (not both)
- In-app events (promote seasonal features)

**Google Play Store**:
- Update description frequently (Google crawls regularly)
- Long description is searchable (iOS keyword field is not)
- Feature Graphic is prime real estate (high-quality design)
- Promo video auto-plays (make first 3 seconds count)
- Google Play Instant (let users try before installing)
